Char Converter: 2-way Cyrillic and Latin

This is a concept page. It offers simple controls to type in or paste content and convert it using one of the two possible input options:

  • Convert to Latin text typed in Cyrillic using Bulgarian Phonetic Classic keyboard layout.
  • Convert to Cyrillic text typed in Latin: converts according to Bulgarian Phonetic Classic keyboard layout.

Sometimes, writers may be too focused on our own thoughts; and not watch what we actually type on the screen. We might realize that we've been using a wrong language for our input several sentences in. This simple 2-way conversion tool is perfect for such use cases.
