Services

I author, update and maintain technical documentation

Everything about tech docs

I provide tech writing services for the IT community:

  • create text, diagrams, images and video technical content from scratch
  • deploy & maintain documentation portals
  • reorganize your library, wiki, blog or doc web portal
  • help establish tech writing processes in your organization
  • migrate documentation across different platforms and/or formats
  • improve content SEO
  • provide consultancy on content strategy
  • consult you on any tech writing-related matter

Any format, any system, any tool

I can work with any format or toolset. We prefer doc-as-code platforms.

  • good old Word files, DOC, DOCX or ODT
  • XML-based WYSIWYG authoring tools, like Madcap Flare
  • doc-as-code platforms, like MkDocs, Slate, Docusaurus, Jekyll
  • web portals, such as Zendesk, Helpdesk, Hubspot
  • wikis, including Confluence and Wikimedia
